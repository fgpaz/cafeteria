# Costos Fijos y Variables

> [!info] Base de calculo
> Costos estimados por local, salvo donde se indique "por 2 locales". Moneda: ARS. Fecha de estimacion: febrero 2026.

## Costos fijos mensuales (por local)

| Concepto | Estimacion mensual (ARS) | Notas |
|---|---|---|
| Alquiler | $800.000 - $1.200.000 | Local comercial 80-120 m2 en Dorrego/Villa Nueva |
| Sueldos con cargas (6 personas) | $9.000.000 | 6 empleados x $1.500.000 c/u (incluye cargas sociales) |
| Servicios (luz, gas, agua) | $200.000 - $400.000 | Consumo gastronomico con horno y maquina espresso |
| Internet y telefonia | $50.000 - $80.000 | Conectividad para POS y wifi de gestion |
| Seguros (local + responsabilidad civil) | $100.000 - $200.000 | Seguro comercial basico |

| Sistema POS y software | $50.000 - $100.000 | Suscripcion mensual |
| Mantenimiento y limpieza | $100.000 - $200.000 | Productos de limpieza, reparaciones menores |
| **Total costos fijos por local** | **$10.450.000 - $11.480.000** | |
| **Total costos fijos 2 locales** | **$20.900.000 - $22.960.000** | |

> [!info] Peso de la nomina
> Los sueldos representan entre el 78% y 86% de los costos fijos. Es el rubro mas pesado y el que define la estructura de costos del negocio. Cualquier ajuste en dotacion impacta directamente en el punto de equilibrio.

## Costos variables mensuales (por local, a madurez)

Se calculan sobre la base de ventas a madurez: 65 clientes/dia x $13.000 ticket x 30 dias = ~$25.350.000/mes por local.

## Estructura de costos consolidada (2 locales, a madurez)

| Componente | Mensual (ARS) | % sobre ventas totales |
|---|---|---|
| Ventas brutas (2 locales) | ~$50.700.000 | 100% |
| Costos fijos totales | ~$21.900.000 | ~43% |
| Costos variables totales | ~$17.240.000 | ~34% |
| **Costos totales** | **~$39.140.000** | **~77%** |
| **Resultado operativo bruto** | **~$11.560.000** | **~23%** |

> [!info] Lectura del margen
> El margen operativo bruto de ~23% sobre ventas esta dentro del rango objetivo (22-28%). El COGS controlado (25-28%) es posible gracias a menu corto, estandarizacion de recetas y rotacion estacional. Los primeros meses seran deficitarios por la rampa de ventas contra costos fijos estables.

## Sensibilidad de costos

| Variable | Cambio | Impacto mensual (2 locales) |
|---|---|---|
| COGS sube 5 p.p. (de 26% a 31%) | +5% de ventas | +$2.535.000/mes |
| Se agrega 1 empleado por local | +$3.000.000 fijos | +$3.000.000/mes |
| Alquiler sube 20% | +$200.000-$240.000 por local | +$400.000 - $480.000/mes |
| Ticket baja 10% (de $13.000 a $11.700) | -10% de ventas | -$5.070.000/mes en ingresos |

## Costos que escalan con apertura

Algunos costos son compartidos entre ambos locales y no se duplican:

| Concepto | Comportamiento |
|---|---|
| Contabilidad y asesoria legal | Se comparte (un estudio para ambos locales) |
| Marketing estrategico | Se comparte parcialmente (campanas unificadas) |
| Sistema POS | Licencia puede cubrir multiples puntos de venta |
| Sueldos de socia operativa | No incluido en dotacion por local (costo societario) |
